MonetStack: An Android application for artworks searching

MonetStack is an Android app for personalized art discovery, developed using Jetpack Compose and AWS. It fetches data from the Art Institute of Chicago API, stores user preferences in a MongoDB database, and manages users with AWS Cognito. The app also features a serverless contact form emailer for enhanced user engagement.

My Work

  • Developed an Android application for users to search for artworks, update preference and view favorite artworks using Jetpack Compose and Amazon Web Services (AWS).
  • Utilized Retrofit to handle HTTP requests and instantly fetch paginated data from Art Institute of Chicago API.
  • Persisted users' preference by using a paging MongoDB database running in a docker container and deployed to AWS via Lightsail.
  • Implemented Quarkus microservice with complete CRUD operations compiled to jvm-jar and running in a docker container deployed to AWS via Lightsail on same Lightsail instance as the MongoDB.
  • Integrated user management (registration, login) using AWS Cognito User Pools.
  • Created a serverless contact-form emailer by utilizing AWS Lambda, integrated it with the Android application and deployed it to AWS via SAM and configured to use API-Gateway and SES (Amazon Simple Email Service).
